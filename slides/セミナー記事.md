# Claude Code はじめてガイド
### 〜 AIを「使う」から「動かしてもらう」へ 〜

> 📋 **この記事について**
> - 対象：AIツールをはじめて使う方
> - 所要時間：約65分
> - 使用ツール：Claude Code（デスクトップアプリ版）

---

## AIって何？Claudeって何？

AIは**あなたの仕事を手伝ってくれる賢いアシスタント**です。このセミナーでは、AIをはじめて使う方でも「頼める」ようになることを目指します。

今日使うのは **Claude（クロード）** というAIです。**Anthropic（アンソロピック）** が開発した、安全で信頼できるAIアシスタントです。

---

## PART 1：Claudeの種類を知ろう

### チャット型AIとAIエージェントの違い

ChatGPTのように「質問すると答えが返ってくる」のが **チャット型AI** です。今日学ぶのは **AIエージェント（＝代わりに動いてくれるAI）** です。

| | チャット型 | AIエージェント型（今日学ぶもの） |
|---|---|---|
| 代表例 | ChatGPT・ブラウザ版 claude.ai | **Claude Code** |
| できること | 質問 → 答えが返ってくる | 「○○して」→ 最後までやってくれる |
| ファイル操作 | ❌ できない | ✅ 読む・書く・保存まで全部 |
| イメージ | 賢い辞書・検索エンジン | 実際に動いてくれる秘書 |

**「答えを教えてもらう」から「やってもらう」へ。** これが今日の一番大事なポイントです。

---

### 3つのプラットフォーム

| | ブラウザ版 | **デスクトップアプリ版** ★今日はこれ | ターミナル型 |
|---|---|---|---|
| 使い方 | ブラウザで開くだけ | PCにアプリをインストール | 黒い画面（コマンド入力） |
| 向いている人 | 誰でも | **一般ビジネスユーザー** | エンジニア |
| ファイルアクセス | ❌ 基本できない | ✅ PCのファイルに直接アクセス | ✅ PCのファイルに直接アクセス |
| AIエージェント | △ 限定的 | ✅ 使える | ✅ 使える（上級者向け） |

今日は**デスクトップアプリ版**を使います。マウスとキーボードで操作でき、PCのファイルにも直接アクセスできます。

---

### Claude Cowork と Claude Code の違い

| | Claude Cowork | Claude Code（今日学ぶもの） |
|---|---|---|
| 何者か | 業務ワークフロー自動化に特化 | 汎用AIアシスタント（何でも頼める） |
| リリース状況 | 試験運用中 | 正式リリース済み |
| ファイルアクセス | 範囲に制限あり | PCのファイルすべてにアクセス可 |
| できることの広さ | 特定の業務フローに強い | 幅広い業務に対応 |
| おすすめ度 | △ 今は様子見 | ✅ 本格的に使うならこちら |

Claude Cowork でできることは **Claude Code でも全部できます**。迷ったら **Claude Code 一択** です。

---

### デスクトップアプリ版をおすすめする理由

**① ルールをずっと覚えてくれる**
「丁寧な敬語を使って」「件名は簡潔に」といったルールを最初に書いておくだけで、その後ずっと守ってくれます。

**② 変更前に確認できる**
AIが何かを実行する前に「これでよいですか？」と確認してくれます。やり直しもできるので安心です。

## PART 2：使う前の準備

### インストール手順

> 📸 **[スクリーンショット]** claude.com/download を開いた状態のブラウザ画面

**STEP 1：** ブラウザで `claude.com/download` にアクセス

**STEP 2：** 「Download for Mac」または「Download for Windows」をクリック

**STEP 3：** ダウンロードしたファイルを開いて、画面の指示に従ってインストール

> 📸 **[スクリーンショット]** インストール完了画面 → アプリが起動した直後の画面

---

### 最初にやっておきたい設定（1つだけ）

**「データ学習」をオフにする設定**です。入力した文章がAIの学習に使われるのを防ぎます。**業務で使う場合はオフにしておきましょう。**

> 📸 **[スクリーンショット]** claude.ai/settings/data-privacy-controls の画面。トグルを赤丸で囲む

**STEP 1：** `claude.ai/settings/data-privacy-controls` にアクセス

**STEP 2：** 「Allow Anthropic to use my data for model training」をオフにする

これだけで完了です。

---

## PART 3：話しかけてみよう

### まずは起動してみましょう

> 📸 **[スクリーンショット]** Claude Desktop App を起動した直後のホーム画面全体

アプリを開くと、LINEのようなチャット画面が表示されます。

> 📸 **[スクリーンショット]** テキスト入力欄に「こんにちは」と入力している状態

「こんにちは」と入力して送ってみてください。それだけでClaudeとの会話が始まります。

---

### ファイルを指定して頼んでみましょう

**「どのファイルを使うか」を伝えるだけで、そのファイルを読んで作業してくれます。**

```
demo フォルダの「会議メモ_元データ.md」を読んで、
正式な議事録にまとめてください。
決定事項と担当者・期日が一目でわかる形にしてください。
```

> 📸 **[スクリーンショット]** 上の入力文を打ち込んでいる画面

> 📸 **[スクリーンショット]** Claude が議事録を出力している画面（完成後）

**「このファイルを使って、こうしてほしい」と伝えるだけ**で作業が完了します。

---

### 気に入らなければ、追加で伝えるだけ

結果が気に入らなくても、**追加で指示を出すだけで何度でもやり直せます。**

- 「もう少し短くしてください」
- 「アクションアイテムを表形式にしてください」

Claudeは会話の流れを覚えているので、最初から説明し直す必要はありません。

## PART 4：こんなことができます

### 文章を作ってもらう

```
取引先の田中部長へのお礼メールを書いてください。
先日の打ち合わせで新商品の提案を前向きに検討してもらいました。
丁寧なビジネス文体でお願いします。
```

**宛先・用件・雰囲気**の3つを伝えるだけで、すぐに使えるメール文が完成します。

> **演習（3分）：自分の状況に合わせたメールを書いてもらいましょう**
> 今送りたいメールを思い浮かべて、宛先と用件を入力してClaudeに作ってもらいましょう。

---

### 資料を整理してもらう

```
demo フォルダの「報告書_下書き.md」を読んで、
きちんとしたビジネス報告書に書き直してください。
見出しと箇条書きを使って読みやすくしてください。
```

下書きをClaudeに渡せばきちんとした体裁に整えてくれます。

---

### アイデアを出してもらう

```
社員30人の会社で、チームの結束力を高めるための
社内イベントのアイデアを5つ出してください。
予算は一人3,000円以内、業務時間内でできるものでお願いします。
```

**「何人規模」「予算」「どんな場で使うか」**など、条件を具体的に伝えるほど使えるアイデアが返ってきます。

> **演習（5分）：自分の業務テーマでアイデアを出してもらいましょう**
> 頭を悩ませているテーマを思い浮かべて、条件を入力してClaudeに提案してもらいましょう。

## PART 5：安心して使うコツ

### 作業前に確認できる：/plan

**`/plan`** と入力すると、Claudeが作業前に「こういう手順でやります」と教えてくれます。

> 📸 **[スクリーンショット]** チャット欄に「/plan」と入力 → 作業手順を箇条書きで提案している画面

1. お願いしたい内容の前に **`/plan`** と入力
2. Claudeが手順を箇条書きで提案
3. 「OK」で作業開始、「やめて」で中止

---

### 元に戻せる：/rewind

**`/rewind`** と入力すると、直前の状態にファイルを戻せます。ゲームの「セーブデータを読み込む」感覚です。

Claudeは作業のたびに**チェックポイントを自動で記録**しています。「やり直したい」と思ったら `/rewind` と入力するだけです。

---

### CLAUDE.md（クロードエムディ）を育てよう

**CLAUDE.md** は、Claudeに毎回伝えたいことをまとめて書いておくファイルです。次回からClaudeが自動で読み込む「指示書」です。

```
# 私のCLAUDE.md（例）

- 私は不動産会社の営業担当です
- 文体は丁寧な敬語でお願いします
- お客様向けの文章は「です・ます」調にしてください
```

使いながら少しずつ育てていくと、あなた専用のアシスタントになっていきます。

## 付録 B：よくある質問

**Q1. 入力した内容は外に漏れますか？**
Anthropicのサーバーに送信されます。業務の機密情報は入力しないか、社内ポリシーを確認してください。「データ学習オフ」の設定をしておくと安心です。

**Q2. 間違えたら元に戻せますか？**
はい。`/rewind` で直前の状態に戻せます。

**Q3. 会社のパソコンで使っていいですか？**
社内のITポリシーを確認してから使ってください。

**Q4. 費用はかかりますか？**
有料プラン（Pro）が必要です。月額 $20（約3,000円）です。
