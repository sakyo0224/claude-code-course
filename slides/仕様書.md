#### スライド 1-0：みなさんが知っているAIと、今日学ぶAIの違い

> **前提（セミナー運営メモ）：**  
> 参加者はChatGPTなどのチャット型AIをすでに知っている。  
> AIエージェントという概念は知らない。「チャット型との違い」を起点に説明する。

> **補足説明（本文に入れる）：**  
> 「AIエージェント」とは、AIが自分で考えて**実際に動いてくれる**仕組みのことです。  
> エージェント＝「代理人」。あなたの代わりに手を動かしてくれます。  
>  
> みなさんが知っているChatGPTやclaude.aiは「チャット型」です。  
> 今日学ぶ Claude Code は、それより一段階上の「AIエージェント型」です。

```
🖼️ [NanoBanana]
左側に「チャット型（みんなが知ってるやつ）」：人間が質問→AIが答えを返す一問一答（吹き出し1往復のシンプルな図）、
右側に「AIエージェント型（今日学ぶやつ）」：人間が一言頼む→AIが複数の作業（読む・書く・保存）を自律的にこなす（矢印が連鎖する図）、
左右を対比したレイアウト、右側が輝いて強調、フラットイラスト
```

| | チャット型（知ってるやつ） | AIエージェント型（今日学ぶやつ） |
|---|---|---|
| 代表例 | ChatGPT・ブラウザ版 claude.ai | **Claude Code** |
| できること | 質問 → 答えが返ってくる | 「○○して」と頼む → 最後まで**やってくれる** |
| ファイル操作 | ❌ できない | ✅ 読む・書く・保存まで全部 |
| イメージ | 賢い辞書・検索エンジン | **実際に動いてくれる秘書** |

> **セミナーでの一言：**  
> 「答えを教えてもらう」から「やってもらう」へ。それが今日の一番大事なポイントです。

### B. よくある質問（FAQ）

**Q. 入力した内容は外に漏れますか？**  
→ Anthropic のサーバーに送信されます。業務の機密情報は入力しないか、社内ポリシーを確認してから使ってください。PART 2 で行った「データ学習オフ」の設定をしておくと安心です。

**Q. 間違えたら元に戻せますか？**  
→ はい。`/rewind` と入力するだけで直前の状態に戻せます。

**Q. 会社のパソコンで使っていいですか？**  
→ 社内の IT ポリシーやセキュリティガイドラインを確認してから使ってください。

**Q. 費用はかかりますか？**  
→ Claude の有料プラン（Pro）が必要です。月額 $20（約3,000円）です。
