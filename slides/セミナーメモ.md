# ClaudeCodeセミナー 講師メモ

## セミナー概要

- **対象:** プログラミング未経験のビジネスパーソン
- **目標:** ClaudeCodeを使って「文章・資料・アイデア出し」の仕事を効率化できるようになる
- **所要時間:** 90分

---

## アジェンダ

### 第1部: ClaudeCodeって何？（15分）

**話す内容:**

ClaudeCodeは「ターミナル（黒い画面）で使うAIアシスタント」。
ChatGPTと何が違うの？という疑問には：

| | ChatGPT（ブラウザ） | ClaudeCode（ターミナル） |
|---|---|---|
| 使い方 | ブラウザで会話 | 自分のPCで直接作業 |
| ファイル操作 | できない | できる（読む・書く・保存） |
| 自分のルール | 毎回伝える必要あり | CLAUDE.mdに一度書けば覚えてくれる |
| 自動化 | できない | Hooksで自動処理できる |

**伝えるポイント:**
- ファイルを直接読んで、直接編集・保存してくれる
- 「毎回同じことを説明しなくていい」のが最大の強み
- プログラミングの知識は不要

**デモ:**
1. ターミナルで `claude` と入力して起動
2. 「自己紹介してください」と話しかけてみる
3. `demo/会議メモ_元データ.md` を読ませて議事録にしてもらう

---

### 第2部: CLAUDE.md — 「お約束ごと」ファイル（15分）

**話す内容:**

CLAUDE.mdは「このフォルダでの約束ごとをClaudeCodeに伝えるファイル」。

例えば：
- 「いつも丁寧な敬語で書いて」
- 「私の会社はXX業界なので、専門用語はXXを使って」
- 「必ず日本語で答えて」

→ 一度書いておけば、毎回言わなくていい！

**デモ:**
1. このフォルダの `CLAUDE.md` を開いて見せる
2. CLAUDE.mdに新しいルールを追加してみる（例：「文末は必ず「です・ます」調にする」）
3. 追加後にClaudeCodeに何か書かせて、ルールが効いていることを確認

**参加者に考えてもらう:**
> 「あなたの仕事でClaudeCodeに毎回伝えることは何ですか？」

### 第4部: もっと活用するために（10分）

**紹介するポイント:**

#### 自分だけのCLAUDE.mdを育てよう
```
# 私の仕事用CLAUDE.md（例）

- 私は不動産会社の営業担当です
- 文体は丁寧な敬語でお願いします
- 数字は必ず「万円」単位で表記してください
- お客様向け文書は「です・ます」調、社内向けは「だ・である」調にしてください
```

#### Hooks（自動処理）の紹介
「ファイルを保存するたびに自動でログを残す」などが設定できる。
→ 今回は設定済み。`/tmp/claude_seminar.log` に記録されている。

#### こんな使い方もできます
- 長い議事録から「自分の担当タスクだけ」を抜き出してもらう
- 英語のメールを日本語に訳してから返信文を作ってもらう
- 複数の資料を読ませて比較表を作ってもらう
- 毎週の週報テンプレートをフォルダに入れておいて自動で埋めてもらう

## よくある質問

**Q: プログラミングがわからなくても使えますか？**
A: はい、普通の日本語で話しかけるだけで大丈夫です。

**Q: 入力した情報はどこかに漏れませんか？**
A: 入力内容はAnthropicのサーバーに送られます。社外秘の情報は入力しないよう注意してください。

**Q: ChatGPTと使い分けるとしたら？**
A: ブラウザでさっと聞きたいならChatGPT。ファイルを直接操作したい・ルールを覚えさせたいならClaudeCode。

**Q: 料金はかかりますか？**
A: API利用分の費用がかかります。一般的な業務利用（1日数時間）で月数千円〜1万円程度の目安。
