# Claude Code でできること — 全体像ガイド
### プログラミングなしで、あなたの仕事はここまで変わる

---

## はじめに — Claude Code とは何か

Claude Code は、**自分のパソコン上で動くAIアシスタント**です。

ChatGPT や Copilot と似ていますが、決定的に違う点が一つあります。

> **ファイルを直接読んで、編集して、保存できる。**

ブラウザの会話画面に貼り付けて使うのではなく、あなたのパソコンにある実際のファイル——Excel の下書き、会議メモ、顧客リスト——を直接扱えます。

---

## 「今、何に困っているか」から考える

Claude Code が役立つ場面を、業務の具体例で見てみましょう。

| こんな作業、ありませんか？ | Claude Code を使うと |
|--------------------------|---------------------|
| 会議の録音メモを議事録にまとめるのに毎回30分かかる | 「このメモを議事録にして」の一言で完成 |
| 月次レポートをExcelから手作業でWord に転記している | CSVを渡すだけでレポート自動生成 |
| 50社へのご案内メールを1通ずつ手直しして送っている | 顧客リストから個別メールを一括作成 |
| 競合他社の情報をウェブで調べて比較表を作っている | 調査から表の作成まで自動でこなす |
| 同じ指示を毎回AIに説明しなおしている | 一度設定すれば、次から不要 |
| 大量のPDFから必要な情報だけ抜き出したい | ファイルを渡すだけで抽出・整理 |

## 成長の4ステージ — コース全体の地図

このコースは60本の記事で構成されており、4つのステージで段階的に力をつけていきます。

```
【ステージ1】 はじめの一歩
     ↓  インストールして、最初の会話ができる
【ステージ2】 日常業務に組み込む
     ↓  ファイル操作・ルール設定・変更管理をマスター
【ステージ3】 仕組みで動かす
     ↓  自動化・外部サービス連携・定期実行
【ステージ4】 組織に広げる
        チーム導入・品質管理・大規模運用
```

## ステージ1：はじめの一歩（15記事）

**ゴール：** 安心して使い始められる

### この段階でできるようになること

- Claude Code を自分のパソコンにインストールして起動できる
- 文章を渡して「要約して」「メールに書き直して」などの指示ができる
- 写真やスクリーンショットを見せて内容を説明してもらえる
- 料金の仕組みを理解し、無駄なコストを避けられる

### 身近な活用イメージ

```
例：取引先から届いた長い提案書（PDF）を要約させる
例：部下が書いた報告書の下書きを、丁寧な敬語に整える
例：英語のニュースリリースを日本語に翻訳してもらう
```

---

## ステージ2：日常業務に組み込む（16記事）

**ゴール：** 繰り返し作業を Claude Code に任せられる

### この段階でできるようになること

- フォルダ内のファイルを読み込み・編集・保存させられる
- 「いつも敬語で」「必ず日本語で」などのルールを一度設定すれば毎回自動適用
- 大きな作業を計画的に進め、途中で確認しながら進める
- 変更内容を確認して、必要なら元に戻せる
- データ分析レポート・一括メール生成・競合調査などの実践的な作業

### 身近な活用イメージ

```
例：毎月の売上CSV（5店舗分）を渡すと月次レポートが自動で完成
例：名刺交換リスト（50社）から個別のご案内メールを一括生成
例：競合A社〜D社を調査して、比較表と考察コメントを作成
```

### 特に重要な「CLAUDE.md」という仕組み

> **一度書いたルールを、Claude Code が毎回覚えていてくれる**

「私の会社は不動産業界です」「報告書は1000字以内で」「語尾は〜です・ます調で」——こういった前提を毎回説明しなくてよくなります。チームで共有すれば、全員が同じルールのもとで使えます。

---

## ステージ3：仕組みで動かす（16記事）

**ゴール：** 「言わなくても動く」仕組みを作れる

### この段階でできるようになること

- ファイルを保存したとき・作業が完了したときに自動で処理が走る（Hooks）
- 「毎朝9時にニュースをまとめて」のような定期実行タスクを設定できる
- Slack・Notion・Googleカレンダーなど外部サービスと連携できる（MCP）
- 複数のAIエージェントに役割分担させて並列作業できる
- スマホや外出先からリモートで操作できる

### 身近な活用イメージ

```
例：毎朝7時に業界ニュースを自動収集→要約→Slackに投稿
例：会議が終わるたびに議事録を自動生成してNotionに保存
例：受注ファイルが届くたびに確認メールの下書きを自動作成
```

---

## ステージ4：組織に広げる（13記事）

**ゴール：** チームや会社全体の生産性を底上げできる

### この段階でできるようになること

- コストを最小化しながら最大の効果を出す使い方の設計
- 複数のAIエージェントがチームとして協調して動く仕組みを構築
- 会社のルール・コンプライアンスに準拠した運用体制の整備
- 大量のドキュメントや業務プロセス全体の自動化設計
- トラブル発生時の原因特定と対処の手順

### 身近な活用イメージ

```
例：部門ごとに異なるルールを設定し、全社で統一した品質で運用
例：新人が入ってきたときのオンボーディング資料作成を自動化
例：月次の監査レポートをシステムが自動生成し、担当者が確認するだけ
```

## よくある疑問

**Q. プログラミングの知識は必要ですか？**
ステージ1〜2は完全に不要です。ステージ3〜4では設定ファイルの編集（テキストを書く程度）が出てきますが、コードを書く必要はありません。

**Q. 英語が必要ですか？**
コースはすべて日本語です。Claude Code 自体も日本語で指示を出せます。

**Q. セキュリティは大丈夫ですか？**
Claude Code は自分のパソコン上で動きます。ファイルを外部に送信するタイミングは自分でコントロールできます。何を許可するかの設定も細かく調整できます。

**Q. ChatGPT との使い分けは？**
| やりたいこと | おすすめ |
|-------------|---------|
| さっと質問したい | ChatGPT（ブラウザ） |
| ファイルを処理したい | Claude Code |
| 繰り返し作業を自動化したい | Claude Code |
| 外部サービスと連携したい | Claude Code |
