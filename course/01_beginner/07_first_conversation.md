# はじめての会話 — テキスト入力の基本

## この記事で学ぶこと
- Claude Codeに指示を出す基本的な方法
- 良い指示（プロンプト）の書き方のコツ
- 指示の出し方による結果の違いを体感する

## 前提知識
- [デスクトップアプリのインストールと初期設定](06_install_desktop.md)（アプリがインストール済み）

## 本文

### 基本的な操作方法

Claude Codeデスクトップアプリを開くと、画面下部にテキスト入力欄があります。ここに日本語で指示を書いて、Enterキーを押すだけで、Claudeが応答してくれます。

操作の流れは非常にシンプルです。

1. 入力欄に指示を書く
2. Enterキーを押して送信
3. Claudeの回答を待つ
4. 必要に応じて追加の指示を送る

### メッセージの送信方法

- **1行のメッセージを送る** ：テキストを入力してEnterキー
- **改行を含むメッセージを送る** ：Shift＋Enterで改行してからEnterで送信
- **送信を取り消す** ：Claudeが回答を生成している途中でEscキーを押すと中断できます

### 良い指示の書き方

Claude Codeに良い結果を出してもらうためのコツをご紹介します。

**コツ1：具体的に書く**

あいまいな指示よりも、具体的な指示の方が良い結果が得られます。

```
× いい感じのメールを書いて
○ 取引先の山田様宛に、来週水曜日の打ち合わせ日程を確認するビジネスメールを書いてください。丁寧な文体でお願いします。
```

**コツ2：役割を指定する**

「あなたは○○です」と役割を伝えると、その立場に沿った回答をしてくれます。

```
あなたは経験豊富なビジネスライターです。
新規事業の提案書の冒頭あいさつ文を書いてください。
読者は経営層です。
```

**コツ3：出力形式を指定する**

欲しい形式を明確に伝えましょう。

```
以下の内容を箇条書きで5つにまとめてください。
```

```
以下の情報を表形式で整理してください。列は「項目」「担当者」「期限」の3つです。
```

**コツ4：条件や制約を伝える**

文字数や言葉遣いなどの条件を伝えると、より意図に近い結果が得られます。

```
以下の報告書を300字以内で要約してください。
専門用語は使わず、新入社員でもわかる表現にしてください。
```

### 会話のキャッチボール

Claude Codeとのやり取りは、1回の指示で終わりではありません。会話のように、やり取りを重ねてより良い結果に近づけることができます。

**例：議事録の作成**

```
あなた：以下の会議メモから議事録を作成してください。
        （メモの内容）

Claude：（議事録を作成）

あなた：決定事項の部分をもう少し詳しく書いてください。

Claude：（決定事項を詳細化）

あなた：最後に「次回のアクション」のセクションを追加してください。

Claude：（アクションセクションを追加）
```

このように、段階的に指示を出して仕上げていくのも有効な使い方です。

### 注意点

- **個人情報や機密情報に注意** ：顧客の個人情報や社外秘のデータを入力する際は、プライバシー設定を確認してください
- **回答は必ず確認する** ：AIの回答が100%正確とは限りません。特に数値やデータは必ずチェックしましょう
- **長すぎる文章は分割する** ：非常に長い指示は、いくつかに分けて送った方が正確な結果が得られることがあります

## やってみよう（ハンズオン）

### ステップ1：簡単な質問をしてみましょう

デスクトップアプリを開いて、以下の文章を入力してEnterキーを押してください。

```
ビジネスメールでよく使う締めの挨拶を5つ教えてください。
```

回答が返ってきたら、内容を確認してみましょう。

### ステップ2：条件をつけた指示を試しましょう

今度は、より具体的な条件をつけて指示してみましょう。

```
新しく異動してきた同僚への歓迎メールを書いてください。
条件：
- 宛名は「佐藤さん」
- 送信者は営業部の田中
- 200字程度
- カジュアルすぎない丁寧な文体
- ランチに誘う内容を含める
```

### ステップ3：会話を続けてみましょう

ステップ2の結果に対して、追加の指示を送ってみましょう。

```
ありがとうございます。この文面をもう少しフォーマルにしてください。
また、部署の簡単な紹介も1文追加してください。
```

指示の出し方によって結果がどう変わるか、違いを観察してみましょう。

## まとめ
- テキストを入力してEnterキーで送信するだけの簡単操作
- 具体的に、役割・形式・条件を指定すると良い結果が得られる
- 会話のやり取りを重ねて、より良い結果に仕上げることができる
- AIの回答は必ず人間が確認することが大切
