# トークン消費を意識する

## この記事で学ぶこと
- 「トークン」とは何かを理解する
- トークン消費を確認する方法を知る
- トークンを節約するコツを身につける

## 前提知識
- [キーボードショートカットで効率アップ](13_keyboard_shortcuts.md)

## 本文

### トークンとは

「トークン」とは、AIが文章を処理する際の単位のことです。日本語の場合、おおよそ1文字が1〜2トークンに相当します。英語の場合は、1単語が1〜2トークン程度です。

たとえば、「今日はいい天気ですね」という文章は、約15〜20トークン程度になります。

Claude Codeを使うと、以下の場面でトークンが消費されます。

- あなたが入力した指示（入力トークン）
- Claudeが生成した回答（出力トークン）
- ファイルの読み込み（ファイルの内容分のトークン）
- 画像の処理（画像のサイズに応じたトークン）

### Proプランの利用制限

Proプラン（月額$20）では、5時間あたり約45メッセージが利用の目安です。ただし、この数字は固定ではなく、メッセージの長さや複雑さによって変わります。

- **短い質問**（「こんにちは」など）：トークン消費が少ない
- **長い指示＋長いファイルの処理** ：トークン消費が多い
- **画像の処理** ：画像1枚あたり約1,334トークン（1000x1000ピクセルの場合）

制限に達すると、一時的にメッセージを送れなくなりますが、時間が経てばまた利用できるようになります。

### トークン消費を確認する方法

**方法1：/cost コマンド**

```
/cost
```

現在のセッションで消費したトークン数が表示されます。

**方法2：/usage コマンド**

```
/usage
```

プラン全体の利用状況と残りの制限が表示されます。

**方法3：/context コマンド**

```
/context
```

現在の会話で使っているコンテキスト（文脈情報）の量を可視化します。会話が長くなりすぎていないか確認するのに便利です。

### トークンを節約する7つのコツ

**コツ1：具体的な指示を出す**

あいまいな指示だと、Claudeが意図をくみ取るために何度もやり取りが必要になります。最初から具体的に伝えましょう。

```
× 何かいい感じに直して
○ 3段落目の文章を、より丁寧な敬語に修正してください
```

**コツ2：/compact で会話を定期的に圧縮する**

会話が長くなると、過去のやり取りすべてがトークンとして消費されます。ひと区切りついたタイミングで圧縮しましょう。

```
/compact
```

**コツ3：話題が変わったら /clear で新しいセッションを始める**

前の話題のトークンが無駄になります。話題が変わったらリセットしましょう。

```
/clear
```

**コツ4：大きなファイルは必要な部分だけ渡す**

100ページの文書を丸ごと渡すよりも、必要な部分だけ抜き出して渡す方がトークンを節約できます。

**コツ5：画像は必要最小限のサイズにする**

スクリーンショットは画面全体ではなく、必要な部分だけをトリミングしてから渡しましょう。

**コツ6：出力の長さを指定する**

「300字以内で」「3行で」のように、回答の長さを指定すると、不要に長い回答によるトークン消費を防げます。

```
この報告書を3行で要約してください。
```

**コツ7：/effort で推論レベルを調整する**

簡単な質問には低い推論レベル、複雑な分析には高い推論レベルを使い分けることで、トークンを効率的に使えます。

```
/effort low
```

### 消費量の目安

目安として、一般的な利用パターンでのトークン消費量をご紹介します。

| 操作 | トークン消費の目安 |
|------|------------------|
| 短い質問と回答 | 数百〜1,000トークン |
| ビジネスメールの作成 | 1,000〜3,000トークン |
| 文書の要約（1ページ程度） | 2,000〜5,000トークン |
| 画像1枚の処理 | 約1,300トークン追加 |
| 長い文書の分析 | 5,000〜20,000トークン |

### 制限に達したときの対応

利用制限に達した場合は、以下の対応があります。

1. **時間を置く** ：5時間のウィンドウが経過すれば、また利用できるようになります
2. **プランをアップグレードする** ：Max 5x（$100/月）やMax 20x（$200/月）に切り替えると、利用量が増えます
3. **使い方を見直す** ：上記のトークン節約のコツを実践して、効率的に利用しましょう

## やってみよう（ハンズオン）

### ステップ1：現在のトークン消費量を確認しましょう

以下のコマンドを実行して、トークン消費量を確認してみましょう。

```
/cost
```

表示された数字を確認してください。まだあまり使っていなければ、少ない数字が表示されるはずです。

### ステップ2：トークン消費の違いを体感しましょう

まず、短い質問をしてみましょう。

```
1+1は？
```

`/cost` でトークン消費量をチェックします。

次に、長い指示を出してみましょう。

```
日本の47都道府県すべてについて、県庁所在地と名産品を1つずつ教えてください。表形式でお願いします。
```

もう一度 `/cost` でトークン消費量をチェックします。短い質問と長い質問で、消費量がどのくらい違うか比較してみてください。

### ステップ3：/compact を体験しましょう

1. いくつか会話をした後に `/context` で現在のコンテキスト量を確認
2. `/compact` で会話を圧縮
3. もう一度 `/context` で圧縮後のコンテキスト量を確認

圧縮前後でどのくらい変わったか、確認してみましょう。

## まとめ
- トークンはAIが文章を処理する単位で、入力・出力・ファイル読み込みで消費される
- `/cost`、`/usage`、`/context` で消費量を確認できる
- 具体的な指示、`/compact` での圧縮、話題ごとの `/clear` がトークン節約の基本
- Proプランは5時間あたり約45メッセージが目安。制限に達しても時間が経てば復活する
