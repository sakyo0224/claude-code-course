## 本文

### Claude Codeの基本

Claude Codeは、Anthropicが提供する「エージェント型AI」ツールです。前の記事で学んだ「実際に動いてくれる秘書」のような存在で、テキストで指示を出すと、ファイルの作成・編集、情報の検索・整理など、さまざまな作業を実際にやってくれます。

もともとはソフトウェア開発者向けのツールとして生まれましたが、現在ではビジネス文書の作成、データ整理、リサーチなど、開発以外の業務にも幅広く活用されています。

### ブラウザ版（claude.ai）との違い

ブラウザ版のClaudeとClaude Codeには、以下のような違いがあります。

| 項目 | ブラウザ版（claude.ai） | Claude Code（デスクトップアプリ） |
|------|------------------------|----------------------------------|
| ファイルアクセス | アップロードが必要 | パソコン上のファイルに直接アクセス |
| ファイルの保存 | 手動でダウンロード | 直接パソコンに保存 |
| 複数ファイルの処理 | 1つずつアップロード | フォルダごとまとめて処理可能 |
| 外部ツール連携 | 限定的 | 豊富（GitHub、Slack、Notionなど） |
| 作業の自動化 | 不可 | コマンドで自動化が可能 |

一言でまとめると、ブラウザ版は「会話相手」、Claude Codeは「作業パートナー」です。

### Claude Codeでできること

Claude Codeでできることを、大きく分けて紹介します。

**1. ファイル操作**
- ファイルの作成、読み取り、編集
- ファイルやフォルダの検索
- ドラッグ＆ドロップでファイルを渡す
- 変更箇所の確認（差分表示）

**2. 文書作成・編集**
- ビジネスメール、報告書、提案書などの作成
- 既存文書の修正・校正
- 複数の資料を読み込んで要約

**3. 情報の整理・分析**
- データの集計・分類
- 表やリストの作成
- 情報の要約

**4. 画像の理解**
- スクリーンショットの内容を読み取る
- 画像内の文字を認識する
- 画像を見ながら指示に応える

**5. コマンド実行**
- パソコン上でコマンド（命令）を実行
- 繰り返し作業の自動化
- 外部ツールとの連携

### Claude Codeの利用形態

Claude Codeは、以下の3つの方法で利用できます。

1. **デスクトップアプリ** ：パソコンにインストールして使う方法。このコースではこの方法を中心に学びます。
2. **ターミナル（コマンドライン）** ：文字だけで操作する方法。開発者向けです。
3. **IDE連携** ：プログラミング用のソフト（VS Codeなど）と組み合わせて使う方法。

初めての方は、デスクトップアプリから始めるのがおすすめです。画面が見やすく、直感的に操作できます。

なお、Anthropic社は「Claude Cowork」というリサーチ向けのツールも提供していますが、用途が異なるため本コースではClaude Codeに集中します。

### セキュリティについて

Claude Codeはパソコン上のファイルにアクセスできるため、セキュリティが気になる方もいらっしゃるかもしれません。

安心していただくために、主なセキュリティ機能をご紹介します。

- **パーミッションモード** ：ファイルの編集やコマンドの実行前に、毎回確認を求める設定ができます（初期設定）
- **計画モード** ：ファイルの変更を行わず、分析と計画だけを行うモードがあります
- **プライバシー設定** ：データをAIの学習に使うかどうかを自分で選択できます

詳しくは後の記事で改めて解説しますので、ここでは「安全に使うための仕組みが用意されている」ということだけ覚えておいてください。

## やってみよう（ハンズオン）

### ステップ1：Claude Codeの活用場面を考えてみましょう

あなたの普段の業務の中で、Claude Codeに手伝ってもらえそうな作業を3つ書き出してみましょう。

例：
- 毎週の週報メールの下書き
- 会議の議事録を要約してもらう
- お客様への提案書のたたき台を作る

### ステップ2：ブラウザ版との違いを体感するイメージ

ブラウザ版では、たとえば10個のファイルを処理したい場合、1つずつアップロードして結果をダウンロードする必要があります。Claude Codeなら、フォルダを指定するだけで10個のファイルを一気に処理できます。

この違いを、実際にClaude Codeをインストールした後に体験してみましょう。

## まとめ
- Claude Codeは「作業パートナー」としてファイル操作や文書作成を実際に行ってくれるエージェント型AI
- ブラウザ版との最大の違いは、パソコン上のファイルに直接アクセスできること
- ファイル操作、文書作成、情報整理、画像理解など幅広い作業に対応
- セキュリティ機能が充実しており、安全に利用できる
