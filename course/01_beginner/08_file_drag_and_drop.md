# ファイルを渡して処理してもらう

## この記事で学ぶこと

- ドラッグ＆ドロップの使い方を覚える
- @メンションでファイルを指定する方法を知る

## 前提知識

## 本文

### ファイルを渡す3つの方法

Claude Codeにファイルを渡して処理してもらう方法は、大きく3つあります。

1. **ドラッグ＆ドロップ** ：ファイルをアプリの入力欄にドラッグして放す
2. **@メンション** ：入力欄に`@ファイル名`と入力してファイルを指定する
3. **パスを直接指定** ：ファイルの場所（パス）をテキストで伝える

### 方法1：ドラッグ＆ドロップ

最も直感的な方法です。

1. Finder（macOS）またはエクスプローラー（Windows）でファイルを見つける
2. ファイルをClaude Codeの入力欄にドラッグする
3. 入力欄にファイルが添付された表示を確認
4. 指示を書いてEnterキーで送信

この方法は、画像やPDFなど、さまざまな種類のファイルに対応しています。

**対応するファイルの例：**
- テキストファイル（.txt、.csv）
- 文書ファイル（.pdf）
- 画像ファイル（.png、.jpg）
- 表計算やその他のファイル

### 方法2：@メンションでファイルを指定

入力欄で`@`を入力すると、ファイル名の候補が表示されます。

```
@売上報告書.pdf この文書の内容を要約してください。
```

この方法は、作業ディレクトリ内のファイルを素早く指定したいときに便利です。ファイル名の一部を入力すると候補が絞り込まれるので、長いファイル名でも簡単に指定できます。

### 方法3：パスを直接指定

ファイルの場所をテキストで伝える方法です。

```
/Users/tanaka/Documents/報告書.txt を読んで内容を要約してください。
```

この方法は、ファイルの正確な場所がわかっている場合に使います。

### ファイルを渡してできること

ファイルをClaude Codeに渡すと、さまざまな処理を依頼できます。

**要約する**
```
@会議議事録_0401.pdf この議事録の内容を300字以内で要約してください。
```

**内容を分析する**
```
@売上データ.csv このデータから、売上が最も高い月と最も低い月を教えてください。
```

**別の形式に変換する**
```
@連絡先リスト.txt この連絡先リストを、名前・電話番号・メールアドレスの3列の表形式に整理してください。
```

**翻訳する**
```
@english_report.txt この英語の報告書を日本語に翻訳してください。ビジネス文書として自然な日本語にしてください。
```

**複数ファイルを比較する**
```
@提案書_v1.txt と @提案書_v2.txt を比較して、変更点をリストアップしてください。
```

### ファイルの保存先について

Claude Codeが新しいファイルを作成する場合、通常は作業ディレクトリ（Claude Codeを起動したフォルダ）に保存されます。

保存先を指定したい場合は、指示の中で明記しましょう。

```
要約した内容を /Users/tanaka/Documents/要約.txt として保存してください。
```

### 注意点

- **ファイルの編集は確認付き** ：初期設定では、Claude Codeが既存のファイルを編集する前に確認メッセージが表示されます。内容を確認してから許可しましょう
- **大きなファイルは分割を検討** ：非常に大きなファイルは処理に時間がかかったり、トークン（利用量）を多く消費したりします
- **機密ファイルの取り扱い** ：個人情報や社外秘の情報を含むファイルを渡す場合は、プライバシー設定を事前に確認してください

## やってみよう（ハンズオン）

### ステップ1：テスト用のファイルを作成しましょう

まず、テスト用のテキストファイルを作成します。メモ帳やテキストエディタで、以下の内容を「テスト.txt」というファイル名で保存してください。

```
2026年第1四半期 営業報告

1月：売上高 1,200万円、新規顧客 5社獲得
2月：売上高 980万円、新規顧客 3社獲得、既存顧客1社解約
3月：売上高 1,500万円、新規顧客 8社獲得、大型案件2件受注

課題：2月の売上低下の原因分析が必要
目標：第2四半期の売上目標 4,500万円
```

### ステップ2：ファイルをドラッグ＆ドロップしてみましょう

1. 作成した「テスト.txt」をClaude Codeの入力欄にドラッグ＆ドロップ
2. 以下の指示を入力してEnterキーを押す

```
この営業報告の内容を分析して、以下を教えてください。
1. 四半期の合計売上高
2. 月別の新規顧客数の推移
3. 第2四半期に向けての改善提案を3つ
```

### ステップ3：結果をファイルに保存してもらいましょう

分析結果を気に入ったら、以下の指示を送ってみましょう。

```
この分析結果を「分析レポート.txt」というファイル名でデスクトップに保存してください。
```

ファイル作成の確認メッセージが表示されたら、内容を確認して許可しましょう。

## まとめ
- ファイルを渡す方法は「ドラッグ＆ドロップ」「@メンション」「パス指定」の3つ
- ドラッグ＆ドロップが最も簡単で直感的
- 要約、分析、変換、翻訳、比較など、さまざまな処理を依頼できる
- ファイルの編集前には確認メッセージが表示されるので安心
