# 画像を使った指示 — スクリーンショットと写真

## この記事で学ぶこと
- 画像をClaude Codeに渡す方法を覚える
- スクリーンショットを活用した指示の出し方を知る
- 画像認識でできることとできないことを理解する

## 前提知識

## 本文

### 画像を渡す方法

Claude Codeは画像の内容を理解する能力を持っています。画像を渡す方法は3つあります。

**方法1：ドラッグ＆ドロップ**

画像ファイルをClaude Codeの入力欄にドラッグ＆ドロップします。前の記事で学んだファイルの渡し方と同じです。

**方法2：クリップボードから貼り付け**

スクリーンショットを撮った直後など、クリップボードに画像がある場合は、キーボードショートカットで直接貼り付けられます。

- **macOS** ：Ctrl＋V
- **Windows** ：Alt＋V

**方法3：スクリーンショットを撮影して直接貼り付け**

macOSでは、Command＋Shift＋4 でスクリーンショットを撮影し、撮影した画像を入力欄にドラッグするのが便利です。Windowsでは、Windows＋Shift＋S でスクリーンショットを撮影し、Ctrl＋V で貼り付けできます。

### 画像を使ってできること

**1. スクリーンショットの内容を読み取る**

エラーメッセージや設定画面のスクリーンショットを見せて、内容を説明してもらえます。

```
（スクリーンショットを貼り付けて）
この画面に表示されているエラーメッセージの意味を教えてください。
```

**2. 画像内の文字を読み取る（OCR的な使い方）**

名刺、チラシ、看板など、文字が写っている画像からテキストを抽出できます。

```
（名刺の写真を貼り付けて）
この名刺に書かれている情報を読み取って、以下の形式で整理してください。
- 氏名：
- 会社名：
- 部署・役職：
- 電話番号：
- メールアドレス：
```

**3. 表やグラフの内容を読み取る**

印刷された表やグラフの画像から、データを読み取ってテキスト化できます。

```
（表の画像を貼り付けて）
この表のデータをCSV形式のテキストに変換してください。
```

**4. デザインやレイアウトについて相談する**

スライドやチラシのデザインについて、フィードバックをもらうこともできます。

```
（スライドのスクリーンショットを貼り付けて）
このプレゼンスライドのデザインについて改善点を3つ教えてください。
見やすさと情報の伝わりやすさの観点でお願いします。
```

**5. 写真の内容を説明してもらう**

場所の写真や商品の写真など、画像の内容を言葉で説明してもらえます。

```
（写真を貼り付けて）
この写真に写っているものを説明してください。
```

### 画像を使う際の注意点

- **画像の解像度** ：文字を読み取る場合は、できるだけ鮮明な画像を使いましょう。ぼやけた画像では読み取り精度が下がります
- **画像のサイズ** ：大きな画像はトークン（利用量）を多く消費します。必要な部分だけをトリミングしてから渡すと効率的です
- **読み取り精度** ：手書き文字や特殊なフォントは、読み取りの精度が下がることがあります。結果は必ず確認してください
- **個人情報** ：画像に個人情報が含まれている場合は、プライバシーに配慮してください

### スクリーンショットの撮り方（おさらい）

よく使うスクリーンショットの撮り方をまとめます。

**macOS：**
- 画面全体：Command＋Shift＋3
- 選択範囲：Command＋Shift＋4（マウスで範囲を選択）
- ウィンドウ：Command＋Shift＋4、その後スペースキー（ウィンドウをクリック）

**Windows：**
- 画面全体：PrintScreen キー
- 選択範囲：Windows＋Shift＋S（マウスで範囲を選択）
- ウィンドウ：Alt＋PrintScreen

## やってみよう（ハンズオン）

### ステップ1：スクリーンショットを撮って渡してみましょう

1. デスクトップの画面や、何かのアプリの画面のスクリーンショットを撮ります
2. Claude Codeの入力欄に画像を貼り付けます（macOS：Ctrl＋V、Windows：Alt＋V）
3. 以下の指示を入力してEnterキーを押します

```
この画面に何が表示されているか説明してください。
```

### ステップ2：文字を読み取ってもらいましょう

手元に名刺やチラシなど、文字が印刷されたものがあれば、スマートフォンで写真を撮ってパソコンに送り、Claude Codeに渡してみましょう。

もし手元にない場合は、以下のように画面上のテキストをスクリーンショットで撮って試してみてください。

1. ウェブサイトの一部をスクリーンショットで撮る
2. Claude Codeに画像を渡す
3. 以下の指示を送る

```
この画像に書かれているテキストをすべて読み取って、テキストとして書き出してください。
```

### ステップ3：画像について質問してみましょう

スクリーンショットや写真を渡して、自由に質問してみましょう。

```
（画像を貼り付けて）
この画像について気づいたことを3つ教えてください。
```

## まとめ
- 画像はドラッグ＆ドロップまたはクリップボードからの貼り付けで渡せる
- スクリーンショットの読み取り、文字認識、データの抽出など幅広く活用できる
- 鮮明な画像ほど読み取り精度が高い
- 大きな画像はトークンを多く消費するので、必要な部分だけトリミングすると効率的
