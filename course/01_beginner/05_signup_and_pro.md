# アカウント作成とPro契約の手順

## この記事で学ぶこと
- Claudeのアカウントを作成する方法を知る
- Proプランに契約する具体的な手順を理解する
- よくあるトラブルと対処法を把握する

## 前提知識
- [料金プランを理解する](04_plans_and_pricing.md)

## 本文

### 事前に準備するもの

アカウント作成とPro契約に必要なものは、以下の3つです。

1. **メールアドレス** ：Googleアカウントまたは任意のメールアドレス
2. **電話番号** ：SMS（ショートメッセージ）を受信できる携帯電話の番号
3. **クレジットカードまたはデビットカード** ：Pro契約の支払いに使用

### アカウント作成の手順

**ステップ1：claude.aiにアクセス**

ブラウザで https://claude.ai を開きます。

**ステップ2：アカウントを作成**

2つの方法から選べます。

- **Googleアカウントで登録する場合** ：「Continue with Google」をクリックし、Googleアカウントを選択します
- **メールアドレスで登録する場合** ：「Continue with email」をクリックし、メールアドレスを入力します。届いたメールのリンクをクリックして認証を完了します

**ステップ3：電話番号認証（初回のみ）**

電話番号の入力を求められます。

1. 国番号が「+81」（日本）になっていることを確認
2. 電話番号を入力（先頭の0を除く形式。例：090-1234-5678 → 9012345678）
3. SMSで届いた認証コード（数字）を入力
4. 「Verify」をクリック

これでアカウント作成は完了です。無料プランとしてすぐにClaude（ブラウザ版）が使えるようになります。

### Proプラン契約の手順

**ステップ4：設定画面を開く**

1. 画面左下にある自分のイニシャル（アイコン）をクリック
2. 表示されたメニューから「Settings」を選択

**ステップ5：Billingページへ移動**

1. Settings画面の中から「Billing」を選択
2. 「Upgrade plan」をクリック

**ステップ6：プランを選択**

1. 「Get Pro plan」をクリック
2. 請求期間を選択
   - 月額：$20/月
   - 年額：$17/月（年一括払い）

**ステップ7：支払い情報を入力**

1. クレジットカードまたはデビットカードの番号を入力
2. カード名義、有効期限、セキュリティコードを入力
3. 「Subscribe」をクリック

**ステップ8：完了確認**

画面に「Pro」の表示が出れば契約完了です。すぐにClaude Codeを含むすべてのPro機能が使えるようになります。

### よくあるトラブルと対処法

| トラブル | 原因と対策 |
|----------|-----------|
| カード決済が失敗する | 海外決済が制限されている可能性があります。カード会社に連絡して海外決済を有効にしてもらいましょう。予備のカードを用意しておくと安心です |
| 認証メールが届かない | 迷惑メールフォルダを確認してください。数分待っても届かない場合は、別のメールアドレスを試しましょう |
| SMS認証コードが届かない | 国番号（+81）が正しいか確認してください。Wi-Fi通話をオフにすると届く場合があります |
| Googleログインでエラーが出る | ブラウザのポップアップブロックを無効にしてください。別のブラウザで試すのも有効です |

### プライバシー設定の確認

アカウント作成後、プライバシー設定を確認しておきましょう。

1. Settings画面を開く
2. 「Privacy」の項目を探す
3. 「Allow Anthropic to use my data for model training」（データをモデル学習に使用する許可）のトグルを確認

- **オン** にすると、あなたのデータがAIの学習に使われます（データ保持期間は5年）
- **オフ** にすると、学習には使われません（データ保持期間は30日）

業務で機密情報を扱う場合は、**オフにしておくことをおすすめします** 。この設定はいつでも変更可能です。

## やってみよう（ハンズオン）

### ステップ1：アカウントを作成しましょう

上記の手順に沿って、https://claude.ai でアカウントを作成してみましょう。

以下のチェックリストで確認してください。

- [ ] claude.aiにアクセスした
- [ ] メールアドレスまたはGoogleアカウントで登録した
- [ ] 電話番号認証を完了した
- [ ] ログインできることを確認した

### ステップ2：Proプランに契約しましょう

アカウントが作成できたら、Proプランに契約しましょう。

- [ ] Settings → Billing を開いた
- [ ] 「Upgrade plan」をクリックした
- [ ] 支払い情報を入力した
- [ ] 「Pro」の表示を確認した

### ステップ3：プライバシー設定を確認しましょう

- [ ] Settings → Privacy を開いた
- [ ] データ学習の許可設定を確認した（必要に応じてオフに変更）

## まとめ
- アカウント作成にはメールアドレスと電話番号が必要
- Pro契約にはクレジットカード/デビットカードが必要
- 契約完了後、すぐにClaude Codeが利用可能になる
- プライバシー設定は早めに確認しておくのがおすすめ
