# キーボードショートカットで効率アップ

## この記事で学ぶこと
- Claude Codeで使えるキーボードショートカットを知る
- よく使うショートカットを覚えて作業を効率化する
- ショートカットのカスタマイズ方法を知る

## 前提知識
- [覚えておきたい基本スラッシュコマンド](12_essential_slash_commands.md)

## 本文

### なぜキーボードショートカットを使うのか

スラッシュコマンドは便利ですが、毎回「/」を入力してコマンド名を打つのは少し手間がかかります。キーボードショートカットを使えば、キーの組み合わせを押すだけで同じ操作を瞬時に行えます。

すべてを覚える必要はありません。自分がよく使う操作のショートカットだけ覚えれば、それだけで作業効率がぐっと上がります。

### 基本操作のショートカット

まずは、毎日使う基本操作のショートカットです。

| ショートカット | 動作 | 説明 |
|----------------|------|------|
| Enter | メッセージ送信 | テキストを入力した後に押して送信 |
| Shift＋Enter | 改行 | メッセージ内で改行したいとき |
| Escape | 入力キャンセル / 中断 | Claudeの回答生成を途中で止めたいとき |
| Ctrl＋C | 操作を中断 | 現在実行中の操作を強制停止 |
| Ctrl＋D | 終了 | Claude Codeを終了する（ターミナル版） |

### 画面操作のショートカット

画面の表示を制御するショートカットです。

| ショートカット | 動作 | 説明 |
|----------------|------|------|
| Ctrl＋L | 画面再描画 | 画面表示が乱れたときにリフレッシュ |
| Ctrl＋T | タスクリスト表示 | バックグラウンドで動いているタスクの一覧を表示 |
| Ctrl＋O | 詳細表示切替 | Claudeの内部処理の詳細を表示/非表示 |

### 作業効率化のショートカット

作業をスピードアップするショートカットです。

| ショートカット | 動作 | 説明 |
|----------------|------|------|
| Ctrl＋R | 履歴検索 | 過去に入力した内容を検索して再利用 |
| Ctrl＋B | バックグラウンドに送る | 時間のかかるタスクを裏で実行させる |
| Ctrl＋G | 外部エディタで開く | 長い入力をエディタで編集したいとき |
| Ctrl＋S | プロンプトをスタッシュ | 入力中の内容を一時保存 |

### 画像とファイルのショートカット

| ショートカット | 動作 | 説明 |
|----------------|------|------|
| Ctrl＋V（macOS） | 画像貼り付け | クリップボードの画像を入力欄に貼り付け |
| Alt＋V（Windows） | 画像貼り付け | クリップボードの画像を入力欄に貼り付け |

### モード切り替えのショートカット

| ショートカット | 動作 | 説明 |
|----------------|------|------|
| Shift＋Tab | パーミッションモード切替 | 確認モード/自動モードなどを切り替え |
| Cmd＋P（macOS）/ Meta＋P | モデル選択 | AIモデルを素早く切り替え |
| Cmd＋T（macOS）/ Meta＋T | 拡張思考切替 | より深く考えるモードのオン/オフ |
| Meta＋O | 高速モード切替 | 速度優先モードのオン/オフ |

### 巻き戻しのショートカット

| ショートカット | 動作 | 説明 |
|----------------|------|------|
| Escape → Escape（2回押し） | 巻き戻し画面 | `/rewind` と同じ機能を呼び出す |

### まずはこの5つから覚えましょう

すべてのショートカットを一度に覚える必要はありません。まずは以下の5つだけ覚えてみてください。

1. **Enter** ：メッセージ送信
2. **Escape** ：中断
3. **Shift＋Tab** ：パーミッションモード切替
4. **Ctrl＋V / Alt＋V** ：画像貼り付け
5. **Escape×2回** ：巻き戻し

この5つだけでも、日常的な操作がかなりスムーズになります。

### ショートカットのカスタマイズ

慣れてきたら、自分好みにショートカットをカスタマイズすることもできます。

```
/keybindings
```

このコマンドで、キーボードショートカットの設定画面を開けます。よく使う操作に自分が押しやすいキーを割り当てることができます。

## やってみよう（ハンズオン）

### ステップ1：基本ショートカットを試してみましょう

以下の操作を順番にやってみてください。

1. テキストを入力して **Enter** で送信

```
こんにちは
```

2. Claudeが回答を生成し始めたら、すぐに **Escape** を押して中断

3. もう一度テキストを入力して、今度は回答を最後まで待つ

```
1から10まで数えてください。
```

### ステップ2：画像貼り付けを試してみましょう

1. デスクトップのスクリーンショットを撮る
   - macOS：Command＋Shift＋3
   - Windows：Windows＋Shift＋S

2. Claude Codeで画像を貼り付ける
   - macOS：Ctrl＋V
   - Windows：Alt＋V

3. 指示を入力して送信

```
この画面に何が表示されていますか？
```

### ステップ3：パーミッションモードの切り替えを試しましょう

1. **Shift＋Tab** を押してみましょう
2. 現在のモードが切り替わることを確認
3. もう一度 **Shift＋Tab** を押して元のモードに戻す

パーミッションモードの表示がどこに出るか、確認してみてください。

## まとめ
- キーボードショートカットを使うと、操作が格段に速くなる
- まずは「Enter」「Escape」「Shift＋Tab」「Ctrl＋V」「Escape×2」の5つから
- 慣れてきたら `/keybindings` でカスタマイズも可能
- すべてを覚える必要はなく、よく使う操作だけ覚えればOK
