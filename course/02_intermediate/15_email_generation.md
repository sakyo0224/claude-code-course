## 本文

### ビジネスシナリオ

あなたは、営業部門でセミナーの集客を担当しています。来月開催するセミナーの案内メールを、過去に名刺交換した50社に送る必要があります。

すべての会社に同じ文面を送るのではなく、各社の業種や前回の接点に合わせて、少しずつ内容を変えた個別メールを作りたいと考えています。

1通ずつ手作業で書くと1通あたり5〜6分、50通で4〜5時間かかる計算です。Claude Code を使えば、この作業を大幅に短縮できます。

### メール一括生成の仕組み

メール一括生成は、「差し込み印刷」と似た考え方です。

1. **顧客リスト（CSV）**：宛先の情報（会社名、担当者名、業種など）
2. **メールテンプレート**：メールの雛形（差し替え箇所を変数として記載）
3. **生成ルール**：文面をどうパーソナライズするかの指示

この3つを Claude Code に渡すと、顧客リストの各行に対応した個別メールが一括生成されます。

### パーソナライズの3段階

メールのパーソナライズには、以下の3つの段階があります。

#### レベル1：名前の差し替え

最もシンプルなパーソナライズです。テンプレートの「{会社名}」「{担当者名}」を、リストの情報で機械的に置換します。

```
{担当者名}様

いつもお世話になっております。
{会社名}様には、先日の{前回の接点}の際に大変お世話になりました。
```

#### レベル2：業種に応じた文面変更

業種や部門に応じて、メールの一部分を変えます。

```
IT企業向け → 「DX推進における最新のAI活用事例をご紹介いたします」
製造業向け → 「製造現場の生産性向上に貢献するAI活用事例をご紹介いたします」
小売業向け → 「顧客体験の向上を実現するAI活用事例をご紹介いたします」
```

#### レベル3：完全にカスタマイズされた文面

各社の状況に応じて、文面全体を個別にカスタマイズします。Claude Code の強みはこのレベル3にあり、単純な差し替えでは実現できない自然な文面を生成できます。

### メール生成の指示のコツ

**1. テンプレートに「トーン」の指示を含める**

```
トーン：丁寧かつ親しみやすい。営業的すぎない。相手の業務に役立つ情報提供の姿勢。
```

**2. 文字数の目安を指定する**

```
本文は300〜400文字程度にまとめてください。長すぎるメールは読まれにくいです。
```

**3. 件名も個別に生成する**

```
件名は20文字以内で、受信者が開封したくなる内容にしてください。
```

### 生成後の確認ポイント

一括生成されたメールは、送信前に必ず確認しましょう。特に以下の点をチェックしてください。

- **会社名・担当者名の間違い**：最も致命的なミスです
- **敬称の抜け**：「様」「御中」が正しく付いているか
- **事実関係**：前回の接点や業種に関する記述が正しいか
- **文面の自然さ**：AIが生成した文面が不自然でないか

### ステップ2：1社分を試しに生成する

まず1社分だけ試してみましょう。

```
customer_list.csv の1行目（株式会社テクノフューチャー）の情報と email_template.txt を使って、個別メールを生成してください。

パーソナライズのルール：
- {担当者名}、{前回の接点} は顧客リストの情報で置換
- {業種に合わせた導入文} は、その会社の業種に関連するAI活用の話題を1〜2文で書く
- {業種に合わせたセミナーの魅力ポイント} は、業種に特化した具体的なメリットを箇条書き2〜3項目で書く
- トーンは丁寧かつ親しみやすく
- 本文全体で400文字以内

生成したメールを email_practice/drafts/techno_future.txt として保存してください。
```

### ステップ3：全社分を一括生成する

1社分の品質に問題がなければ、全社分を生成しましょう。

```
customer_list.csv の全10社について、ステップ2と同じパーソナライズルールで個別メールを生成してください。

各メールは email_practice/drafts/ フォルダに、会社名（英語化）をファイル名にして保存してください。

生成完了後、以下のサマリーも作成してください。
- email_practice/generation_summary.csv（会社名、ファイル名、文字数、パーソナライズした業種ポイントの概要を一覧にしたもの）
```

### ステップ4：生成結果を確認する

```
email_practice/drafts/ フォルダ内のすべてのメールファイルを読んで、以下の観点でチェックしてください。

1. 会社名・担当者名が正しいか
2. 敬称（様）が付いているか
3. 業種に合ったパーソナライズがされているか
4. 文字数が適切か（300〜400文字）
5. 不自然な表現がないか

問題があれば、修正してください。
```

## まとめ
- 顧客リスト（CSV）とテンプレートを組み合わせることで、個別メールを一括生成できる
- パーソナライズは3段階（名前差し替え→業種別変更→完全カスタマイズ）があり、Claude Code はすべてに対応
- まず1社分で試してから全件処理に進むのが安全な進め方
- 生成後は必ず会社名・担当者名・敬称・事実関係を確認する
- 50通で4〜5時間かかる作業を30〜40分に短縮できる
