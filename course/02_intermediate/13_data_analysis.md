## 本文

### ビジネスシナリオ

あなたは、全国に5店舗を展開する小売チェーンの管理部門で働いています。毎月、各店舗から送られてくる売上データ（CSV形式）をもとに、月次売上レポートを作成する業務を担当しています。

これまでは Excel を開いて手作業で集計し、Word でレポートを書いていたため、毎月4〜5時間かかっていました。この作業を Claude Code で効率化してみましょう。

### データの準備

まず、分析対象となるサンプルデータを用意します。実際の業務では既にデータがあるはずですが、ここでは練習用のデータを Claude Code に作ってもらいます。

### 分析の進め方

データ分析は、以下の流れで進めると効果的です。

1. **データの確認**：まずデータの中身を把握する
2. **前処理**：必要に応じてデータをクリーニング（整理）する
3. **集計・分析**：目的に合った計算を行う
4. **レポート作成**：結果をわかりやすくまとめる

Claude Code にまかせれば、この4ステップを一気に処理してもらうこともできますし、ステップごとに確認しながら進めることもできます。大切なデータの場合は、ステップごとに確認しながら進めることをおすすめします。

### 指示のコツ

データ分析を Claude Code に依頼するときのコツをいくつかご紹介します。

**1. 分析の目的を明確に伝える**

```
悪い例：「このCSVを分析して」
良い例：「このCSVから、店舗別の月間売上合計と前月比を計算して、売上が低下している店舗を特定してください」
```

**2. 出力形式を指定する**

```
「結果は以下の形式でCSVファイルに出力してください：
- 1列目：店舗名
- 2列目：今月売上（円）
- 3列目：前月売上（円）
- 4列目：前月比（%）」
```

**3. 判断基準を示す**

```
「前月比が90%を下回る店舗を"要注意"、110%を超える店舗を"好調"と分類してください」
```

### レポートのフォーマット

Claude Code にレポートを作成してもらう際は、フォーマット（書式）を指定すると、毎月統一された形式のレポートが得られます。

```
レポートには以下の項目を含めてください。
- レポートタイトル（月次売上レポート YYYY年MM月）
- 全体サマリー（全店合計売上、前月比、前年同月比）
- 店舗別の売上一覧（表形式）
- 注目ポイント（売上が大きく変動した店舗の分析）
- 推奨アクション（改善が必要な点の提案）
```

## やってみよう（ハンズオン）

### ステップ1：プロジェクトフォルダとサンプルデータを作成する

```
デスクトップに data_analysis_practice というフォルダを作成してください。

その中に以下の2つのCSVファイルを作成してください。

1. sales_202603.csv（2026年3月の売上データ）
ヘッダー：日付,店舗名,商品カテゴリ,売上金額,販売数量
データ：3月1日〜31日の各店舗（東京本店、横浜店、大阪店、名古屋店、福岡店）の日別売上データ。商品カテゴリは「食品」「日用品」「衣料品」の3種類。各行に日付、店舗名、カテゴリ、売上金額（10,000円〜500,000円のランダムな値）、販売数量（10〜200のランダムな値）を含めてください。各店舗×各カテゴリ×各日で1行ずつ、合計約450行のデータにしてください。

2. sales_202602.csv（2026年2月の売上データ）
同じ形式で、2月分のデータを作成してください。東京本店と大阪店は3月より10%ほど低い売上に、名古屋店は3月より15%ほど高い売上に設定してください。
```

### ステップ2：データの概要を把握する

```
data_analysis_practice フォルダ内の sales_202603.csv を読んで、以下を教えてください。

1. データの行数と列数
2. 各列のデータの種類（数値、テキストなど）
3. 店舗名の一覧
4. 商品カテゴリの一覧
5. 売上金額の最大値、最小値、平均値
```

### ステップ3：月次分析を実行する

```
sales_202603.csv と sales_202602.csv を両方読み込んで、以下の分析を行ってください。

1. 店舗別の3月売上合計を計算
2. 店舗別の2月売上合計を計算
3. 店舗別の前月比（3月÷2月×100）を計算
4. 商品カテゴリ別の3月売上合計を計算
5. 全店舗の3月売上合計と2月売上合計、前月比を計算

結果は analysis_result.csv として保存してください。
```

### ステップ4：レポートを作成する

```
分析結果をもとに、以下のフォーマットで月次売上レポートを作成してください。ファイル名は monthly_report_202603.md としてください。

フォーマット：
# 月次売上レポート 2026年3月

## エグゼクティブサマリー
（全体の売上概況を3行で要約）

## 全体実績
（全店合計の売上、前月比を表形式で記載）

## 店舗別実績
（各店舗の売上、前月比を表形式で記載。前月比が低い順にソート）

## カテゴリ別実績
（各カテゴリの売上を表形式で記載）

## 注目ポイント
（前月比が大きく変動した店舗について分析と考察）

## 推奨アクション
（改善が必要な点と具体的な提案を箇条書きで記載）
```

### ステップ5：レポートを確認・修正する

作成されたレポートを確認し、必要に応じて修正を依頼しましょう。

```
monthly_report_202603.md を読んで、内容を確認してください。もし改善点があれば教えてください。

また、エグゼクティブサマリーをもう少し具体的な数字を含めた内容に書き直してください。
```

## まとめ
- データ分析は「確認→前処理→集計→レポート」の4ステップで進める
- 分析の目的、出力形式、判断基準を明確に伝えると、精度の高い結果が得られる
- サンプルデータで練習してから実データに適用すると安心
- レポートのフォーマットを CLAUDE.md に書いておけば、毎月同じ形式で自動生成できる
- 月次レポート作成の業務を4〜5時間から15分程度に短縮できる可能性がある
