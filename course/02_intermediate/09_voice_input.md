# 音声入力で手を動かさずに指示する

## この記事で学ぶこと
- Claude Code の音声入力機能の仕組みを理解する
- /voice コマンドで音声入力モードを使えるようになる
- 音声入力を効果的に活用するコツを身につける

## 前提知識
- [コンテキスト管理 — /compact と /context](08_compact_and_context.md)

## 本文

### 音声入力とは

Claude Code には、キーボードで文字を打つ代わりに、マイクに向かって話しかけることで指示を出せる「音声入力」機能があります。

この機能は「Push-to-Talk（プッシュ・トゥ・トーク）」方式で動作します。特定のキーを押している間だけマイクが有効になり、話した内容がテキストに変換されて Claude Code に送信されます。トランシーバーのボタンを押しながら話すイメージです。

### 音声入力が便利な場面

音声入力は、以下のような場面で特に便利です。

- **手がふさがっているとき**：資料を見ながら指示を出す、別の作業をしながら依頼する
- **長い指示を出すとき**：タイピングよりも話したほうが速い場合がある
- **アイデアを口頭で伝えたいとき**：考えながら話すほうが自然な場合
- **タイピングが苦手な方**：キーボード操作が得意でなくても、声で指示できる

### 音声入力の始め方

音声入力モードは、`/voice` コマンドで切り替えます。

```
/voice
```

このコマンドを実行すると、音声入力モードが有効になります。もう一度 `/voice` を実行すると、元のテキスト入力モードに戻ります。

### 操作方法

音声入力モードが有効な状態では、以下の操作が可能です。

| 操作 | 方法 |
|------|------|
| 録音開始 | **Space キー** を押し続ける |
| 録音停止・送信 | **Space キー** を離す |
| 音声入力のオン/オフ切り替え | `/voice` コマンド |

Space キーを押しながら話し、離すと自動的にテキストに変換されて送信されます。

### 対応言語

Claude Code の音声入力は20以上の言語に対応しており、日本語も使用できます。自然な日本語で話しかければ、Claude Code がその内容を認識してテキストに変換します。

### 音声入力のコツ

音声入力の認識精度を上げるために、以下のポイントを意識しましょう。

#### 1. はっきりと話す

早口や小声では認識精度が下がります。普段の会話よりも少しゆっくり、はっきりと話すことを心がけてください。

#### 2. 静かな環境で使う

周囲の騒音が多いと、正しく認識されないことがあります。できるだけ静かな環境で使いましょう。外出先で使う場合は、イヤホンマイクがおすすめです。

#### 3. 句読点や改行は明示する

音声入力では、句読点や改行が自動で入らないことがあります。必要に応じて「まる」「てん」「かいぎょう」などと言葉で伝えると、より正確なテキストになります。

#### 4. 専門用語は確認する

固有名詞や専門用語は、音声認識で正しく変換されないことがあります。送信前にテキストを確認し、必要に応じて手動で修正しましょう。

### 音声入力とテキスト入力の使い分け

| 場面 | おすすめの入力方法 |
|------|------------------|
| 長い説明や依頼 | 音声入力 |
| ファイル名やパスの指定 | テキスト入力 |
| 簡単な「はい/いいえ」の回答 | どちらでも |
| コマンドの入力（/diff, /rewindなど） | テキスト入力 |
| アイデアの壁打ち・ブレスト | 音声入力 |

基本的に、「正確さが求められる部分」はテキスト入力、「大まかな指示や説明」は音声入力、という使い分けがおすすめです。

### マイクのセットアップ

音声入力を使うには、パソコンにマイクが接続されている必要があります。

- **ノートパソコン**：多くの場合、内蔵マイクがあるのでそのまま使えます
- **デスクトップパソコン**：外付けのマイクやヘッドセットが必要です
- **macOS の場合**：初回使用時に「マイクのアクセス許可」を求められます。許可してください
- **Windows の場合**：設定 → プライバシー → マイク でアクセスを許可してください

## やってみよう（ハンズオン）

### ステップ1：音声入力モードを有効にする

まず、音声入力モードを有効にしましょう。

```
/voice
```

画面に音声入力モードが有効であることが表示されるのを確認してください。

### ステップ2：音声で簡単な指示を出す

Space キーを押しながら、以下のようなことを話してみてください。

「今日の日付を教えてください」

Space キーを離すと、話した内容がテキストに変換されて送信されます。正しく認識されているか確認しましょう。

### ステップ3：音声でファイル作成を指示する

もう少し複雑な指示を音声で出してみましょう。Space キーを押しながら話します。

「デスクトップに voice テスト ドット テキスト というファイルを作ってください。内容は、音声入力のテストファイルです、と書いてください」

認識結果を確認し、ファイル名が意図どおりになっているか見てみましょう。もし認識が不正確な場合は、テキスト入力で修正できます。

### ステップ4：テキスト入力に戻す

音声入力を終了するときは、もう一度 `/voice` を入力します。

```
/voice
```

テキスト入力モードに戻ったことを確認してください。

## まとめ
- Claude Code の音声入力は **Push-to-Talk 方式**で、Space キーを押しながら話す
- `/voice` コマンドで音声入力モードのオン/オフを切り替える
- 日本語を含む20以上の言語に対応している
- はっきりと話す、静かな環境で使う、専門用語は確認する、がコツ
- 長い説明は音声入力、正確なファイル名やコマンドはテキスト入力と使い分けるのが効果的
