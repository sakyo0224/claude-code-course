# Claude Code ではじめるAI活用 ── セミナー資料

> [!info] この資料について
> - **対象**：AIツールを初めて使うビジネスパーソンの方
> - **前提スキル**：メールやExcelを日常的に使えるレベルで十分です
> - **関連スライド**：セミナースライド全38枚に対応（各セクションに対応スライド番号を記載）
> - **セミナー時間**：65分（講義＋デモ＋演習）

## 1. 今日のゴール（スライド 1〜3）

このセミナーを終えたあと、以下の3つができるようになることを目指します。

1. **Claude Code デスクトップアプリを自分で起動し、日本語で仕事を頼める**
2. **失敗しても戻せる・事前に確認できるという安心感を持てる**
3. **「明日、手元のファイルを1つ渡して試してみよう」という具体的な行動イメージがある**

> [!tip] ポイント
> 難しい専門知識は必要ありません。「日本語で話しかけるだけ」で使えるツールです。

## 2. AIとClaude Codeとは（スライド 4〜8）

### AIとは何か

AIは「賢いアシスタント」です。人間が日本語で指示を出すと、文章を書いたり、情報を整理したり、アイデアを出してくれたりします。

プログラミングの知識は一切不要です。普段メールを書くのと同じように、やってほしいことを文章で伝えるだけで動いてくれます。

### 「チャット型」と「エージェント型」のちがい

AIツールには大きく分けて2つのタイプがあります。

| | チャット型 | エージェント型（Claude Code） |
|---|---|---|
| **たとえるなら** | レシピを教えてくれる人 | 料理を作ってくれるシェフ |
| **別のたとえ** | カーナビ（道案内） | 自動運転車（運転までしてくれる） |
| **できること** | 質問に答える・アドバイスする | ファイルを読み書き・作業を実行する |
| **ファイル操作** | 自分でコピー＆貼り付けが必要 | 直接ファイルを読んで、結果を保存してくれる |

> [!important] 覚えておきたいこと
> Claude Code は「エージェント型」です。指示を出すだけでなく、**実際にファイルを読んだり書いたりする作業まで代わりにやってくれます**。

### Claude Code デスクトップアプリの特徴

- パソコン上のファイルに直接アクセスできる（読み書き・保存）
- 複数のセッション（チャット画面）を開いて別々の作業を並行できる
- macOS と Windows に対応（2026年4月3日にWindows対応が追加されました）
- ファイルをドラッグ＆ドロップで渡せる

> [!note] ブラウザ版との違い
> ブラウザ（Webサイト）で使うClaude には、パソコン上のファイルに直接アクセスする機能がありません。ファイルを使った作業には、デスクトップアプリ版をお使いください。

### 関連ツール「Claude Cowork」について

Anthropic社は「Claude Cowork」という別のツールも提供しています。

- **Claude Code**：ファイル操作やコマンド実行が中心
- **Claude Cowork**：画面操作（クリック、ページ移動）や業務別のプラグインが特徴

この2つは**用途が異なるツール**です。それぞれ得意な場面が違いますので、今回のセミナーではファイル操作に強い Claude Code に集中します。

> [!warning] 注意
> Claude Cowork は現在リサーチプレビュー（試験運用中）の段階です。Anthropic社は機密データとの併用を推奨していません。

## 3. 準備：インストールと初期設定（スライド 9〜14）

### インストール手順

1. Anthropic の公式サイトにアクセスする
2. Claude Code デスクトップアプリをダウンロードする
3. ダウンロードしたファイルを開いてインストールする
4. アプリを起動し、アカウントにログインする

### 料金プランについて

| プラン | 月額料金 | 含まれるもの |
|---|---|---|
| **Pro プラン** | $20（年払いの場合 月額$17） | Claude Code、ファイル作成・コード実行、無制限プロジェクト、Google Workspace連携 |

> [!note] 料金について
> - 料金は米ドル表記です。日本円での請求額は為替により変動します
> - Pro プランでは、5時間あたり約45メッセージの利用制限があります

### データの学習設定をオフにする（大切な初期設定）

お仕事のデータをAIに送る前に、**必ずデータ学習の設定を確認**してください。

1. アプリの「設定」を開く
2. 「データの利用」または「Privacy」の項目を探す
3. **学習への利用をオフにする**

> [!important] データの扱いについて
> | 設定 | データ保持期間 | 学習への利用 |
> |---|---|---|
> | **オフ（推奨）** | 30日間 | されない |
> | オン | 5年間 | される可能性あり |
>
> 業務データを扱う場合は、**必ずオフにしてください**。
> なお、Team プランや Enterprise プランなどの商用プランでは、そもそも学習には使用されません。

## 4. 安心装備を先に知る：/plan と /rewind（スライド 15〜19）

実際に作業を始める前に、**「やり直し」と「事前確認」の方法**を知っておきましょう。これを知っているだけで、安心して使えるようになります。

### /plan ── 「まず計画を見せて」コマンド

**やること**：Claude Code を読み取り専用モードにして、変更の「計画」だけを提示させます。

- 入力欄に `/plan` と打ってから指示を出すと、**いきなり変更せず、まず「こうしますがよいですか？」と確認してくれます**
- 計画を見てから「OK」か「やめる」を選べます

> [!tip] 使いどころ
> 大事なファイルを扱うときや、結果が心配なときに使いましょう。「まず計画を見せて」と日本語で頼んでも同じ効果があります。

### /rewind ── 「さっきの状態に戻して」コマンド

**やること**：会話やファイルの状態を、好きな時点まで巻き戻します。

- 入力欄に `/rewind` と打つと、過去のチェックポイント一覧が表示されます
- 戻したい時点を選ぶと、「会話とファイルの両方を戻す」「会話だけ戻す」「ファイルだけ戻す」から選べます
- **Escキーを2回押す**ことでも同じ操作ができます

> [!tip] たとえるなら
> - ゲームの「セーブデータ」をロードするようなものです
> - パソコン操作の「Ctrl + Z（元に戻す）」の強化版と考えてください

### その他の便利コマンド

| コマンド | できること |
|---|---|
| `/plan` | 読み取り専用にして計画だけ提示 |
| `/rewind` | 過去の状態に巻き戻す |
| `/clear` | チャット履歴をすべてクリアする |
| `/help` | 使い方の一覧を表示する |

## 5. はじめての体験：話しかけてみよう【演習1】（スライド 20〜23）

### まずは日本語で話しかけるだけ

Claude Code の使い方はとてもシンプルです。入力欄に日本語で「やってほしいこと」を書くだけです。

### やってみましょう（演習1）

1. Claude Code デスクトップアプリを起動する
2. 入力欄に以下のように打ってみる：

```
こんにちは。あなたは何ができますか？簡単に教えてください。
```

3. Enterキーを押して、返事が来るのを待つ

> [!tip] うまくいかないときは
> - インターネットに接続されているか確認してください
> - アプリにログインできているか確認してください
> - それでもうまくいかない場合は、アプリを一度閉じてから再度開いてみてください

### 話しかけるコツ

- **何をしてほしいか**を具体的に書く
- 一度にたくさん頼まず、**1つずつ**お願いする
- うまくいかなければ、**言い方を変えてもう一度頼む**だけでOK

## 6. ファイルを渡して頼む：議事録づくり【演習2】（スライド 24〜28）

### ファイルの渡し方

Claude Code にファイルを渡す方法は2つあります。

1. **ドラッグ＆ドロップ**：ファイルをアプリの画面に直接ドラッグする
2. **ファイル名を指定する**：入力欄にファイルの場所（パス）を書いて伝える

### やってみましょう（演習2）

1. 手元にあるテキストファイル（メモや会議の記録など）を用意する
2. そのファイルを Claude Code の画面にドラッグ＆ドロップする
3. 以下のように指示を入力する：

```
このファイルの内容を読んで、以下の形式で議事録にまとめてください。
- 日時
- 参加者
- 議題
- 決定事項
- 次のアクション
```

4. 出来上がった議事録を確認する

> [!tip] ポイント
> - 出力のフォーマット（形式）を具体的に指定すると、望み通りの結果が得やすくなります
> - 結果に不満があれば「もう少し箇条書きを増やして」「敬語を使って」など追加で指示するだけで修正してくれます

## 7. 仕事に使う3つのパターン【演習3】（スライド 29〜34）

Claude Code は日常業務のさまざまな場面で活用できます。以下の3つのパターンから、お好きなものを1つ選んで試してみましょう。

### パターンA：メール文面の作成

```
以下の内容でビジネスメールの文面を作ってください。
- 宛先：取引先の田中部長
- 目的：来週の打ち合わせ日程の調整
- 候補日：4月8日（水）午後、4月10日（金）午前
- トーン：丁寧で簡潔に
```

### パターンB：資料の整理・要約

```
この資料を読んで、以下の内容で要約してください。
- 要約は300文字以内
- 重要なポイントを3つ箇条書きにする
- 専門用語があれば簡単な言葉に言い換える
```

### パターンC：アイデア出し

```
新商品のネーミングアイデアを10個出してください。
- 商品：オフィス向けの観葉植物レンタルサービス
- ターゲット：30〜40代の会社員
- 条件：覚えやすい、3〜5文字、カタカナまたはひらがな
```

> [!tip] 演習3のポイント
> どのパターンでも共通する「うまく頼むコツ」は以下の3つです。
> 1. **誰に向けたものか**を伝える
> 2. **どんな形式で**ほしいかを指定する
> 3. **条件や制約**があれば明記する

## 8. 自分専用にする：CLAUDE.md（スライド 35〜37）

### CLAUDE.md とは

**CLAUDE.md** は、Claude Code に「毎回守ってほしいルール」を書いておけるファイルです。

> [!tip] たとえるなら
> 新入社員に渡す「引き継ぎ資料」のようなものです。一度書いておけば、毎回同じことを伝えなくて済みます。

### どんなことを書けるか

- **文体の指定**：「です・ます調で書いてください」
- **出力形式**：「箇条書きでまとめてください」
- **注意事項**：「社外秘の情報は含めないでください」
- **よく使う情報**：「私の部署は営業企画部です」

### 書き方の例

```markdown
- 回答は必ず日本語で行うこと
- 専門用語を避け、わかりやすい言葉で説明すること
- 文章は丁寧な「です・ます」調で書くこと
- 提案は箇条書きなど見やすい形式でまとめること
```

### CLAUDE.md の仕組み

- Claude Code を起動したとき、**最初に自動で読み込まれます**
- 複数の場所に置くことができ、階層構造になっています

| 置く場所 | 範囲 | 優先度 |
|---|---|---|
| パソコン全体の設定（`~/.claude/CLAUDE.md`） | すべてのプロジェクトに適用 | 低い |
| プロジェクトのフォルダ直下 | そのプロジェクトに適用 | 中 |
| サブフォルダ内 | そのフォルダ内に適用 | 高い（最優先） |

> [!note] ポイント
> 作業しているフォルダに近い CLAUDE.md ほど優先度が高くなります。まずはプロジェクトフォルダの直下に1つ作ることから始めてみましょう。

## 9. まとめ：明日からの3ステップ（スライド 38）

今日学んだことを活かして、**明日からすぐに始められる3ステップ**です。

### ステップ1：アプリを起動して話しかける

- Claude Code デスクトップアプリを開く
- 「今日のメールの下書きを手伝って」など、簡単なことから始める

### ステップ2：手元のファイルを1つ渡してみる

- 議事録、報告書、メモなど、なんでもOK
- 「要約して」「整理して」「フォーマットを整えて」と頼んでみる

### ステップ3：CLAUDE.md を作って自分専用にする

- よく使う指示を CLAUDE.md に書いておく
- 毎回同じことを伝える手間がなくなる

> [!important] 安心ポイントのおさらい
> - 不安なときは **`/plan`** で計画を先に確認できます
> - 失敗しても **`/rewind`** で元に戻せます
> - いつでも **`/clear`** で最初からやり直せます

## 付録：よくある質問

### Q. 料金はいくらかかりますか？

Pro プランは月額 $20（年払いの場合は月額 $17）です。日本円での請求額は為替レートにより変動します。5時間あたり約45メッセージまで利用できます。

### Q. 会社の機密情報を入力しても大丈夫ですか？

データ学習の設定をオフにすれば、入力内容がAIの学習に使われることはありません（データ保持期間は30日）。さらに安心して使いたい場合は、Team プランや Enterprise プランの利用をご検討ください。これらの商用プランでは、そもそもデータが学習に使用されません。

### Q. Windows でも使えますか？

はい、使えます。2026年4月3日に Windows 対応が追加されました。macOS と Windows の両方でお使いいただけます（Linux には現在対応していません）。

### Q. 間違った結果が出たらどうすればいいですか？

以下の方法で対処できます。

1. **言い方を変えてもう一度頼む** ── 指示を具体的にすると改善されることが多いです
2. **`/rewind` で巻き戻す** ── ファイルへの変更も元に戻せます
3. **`/plan` で事前確認する** ── 大事な作業の前に計画を確認しましょう

### Q. インターネットに接続していないと使えませんか？

はい、Claude Code はインターネット接続が必要です。AIの処理はクラウド上で行われるため、オフラインでは利用できません。
