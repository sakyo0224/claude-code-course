### 各セクションの詳細

#### 1. オープニング・自己紹介（3分 / 講義）
- タイトルスライド
- 今日のゴール（3つ）
- 今日の流れの概観

#### 2. AIとClaude Codeとは（5分 / 講義）
- AIは「賢いアシスタント」
- チャット型 vs エージェント型の比較（料理人のたとえ）
- Claude Code デスクトップアプリの特徴

#### 3. Cowork vs Claude Code 比較（8分 / 講義＋デモ）★新規セクション
- Cowork と Claude Code の概要比較
- **仮想マシンを使わないメリット**：自分のPC上で直接ファイル操作、動作が軽い、ローカルツールとの連携が自在
- **Windowsでの安定性**：Claude Code デスクトップはネイティブアプリ、Coworkはブラウザ内仮想環境のため接続切れリスクあり
- **トークン消費効率**：Coworkは仮想マシン維持にトークンを消費、Claude Codeはやり取りのみでトークンを使うため効率的
- **Coworkのリサーチプレビュー状態のリスク**：機能の追加・変更・廃止が突然起こりうる、業務で依存するにはリスクが高い
- まとめ：初心者にはClaude Codeデスクトップが安定・安心

#### 4. Proプラン契約・インストール・初期設定（18分 / ハンズオン）
- 料金プランの説明（Pro $20/月）
- 【一斉作業】Proプラン契約の手順（画面を映しながら全員で進む）
- 【一斉作業】デスクトップアプリのダウンロードとインストール
- 【一斉作業】アカウントログイン
- 【一斉作業】データ学習オフの設定
- トラブルシューティング用のスライド（よくある問題と対処法）
- 全員の完了確認

> 8名を1人でサポートするため、18分と余裕を持った設計。画面を大きく映しながら一手順ずつ進める。

#### 5. 安心装備：/plan と /rewind（7分 / 講義＋デモ）
- `/plan` の説明と実演
- `/rewind` の説明と実演
- `/clear` の紹介

#### 6. 【演習1】はじめての会話（5分 / 演習）
- Claude Code に日本語で話しかける
- 「こんにちは。あなたは何ができますか？」
- 返事が来たら成功
- 難易度：★☆☆☆☆

#### 7. Claude Code 全機能の俯瞰（10分 / 講義＋デモ）★新規セクション
以下の機能をカテゴリ別に紹介し、代表的なものはデモを挟む：

| カテゴリ | 機能 |
|---|---|
| 基本操作 | テキスト入力、ファイルD&D、画像認識、マルチセッション |
| ファイル操作 | 読み取り、作成、編集、フォルダ横断処理 |
| コマンド実行 | ターミナルコマンド、スクリプト実行 |
| 安全機能 | /plan、/rewind、/clear、パーミッションモード |
| カスタマイズ | CLAUDE.md、settings.json、hooks |
| 高度な機能 | MCP連携、SubAgent、並行タスク |

- 「今日はこの中から基本操作・ファイル操作・安全機能・カスタマイズを体験します」と範囲を明示

#### 8. ファイルを渡して頼む（5分 / 講義＋デモ）
- ファイルの渡し方（D&D / パス指定）
- 議事録作成のデモ

#### 9. 【演習2】ファイルを渡して議事録を作る（7分 / 演習）

#### 10. 仕事に使う3つのパターン（7分 / 講義＋デモ）

#### 15. ハーネスエンジニアリング（10分 / 講義＋デモ）

#### 16. 【演習5】CLAUDE.md を作成する（5分 / 演習）★新規演習

- CLAUDE.md あり/なしで出力の違いを確認
- 難易度：★★★☆☆

#### 17. まとめ・明日からの3ステップ（5分 / 講義）

  1. アプリを起動して話しかける
  2. 手元のファイルを1つ渡してみる

### 設計上のポイント

1. **「聞くだけ」の連続は最大10分**
   - 講義セクションが10分を超える箇所はなく、必ずデモまたは演習を挟む設計にしています

2. **演習は段階的に難易度が上がる**
   - 演習1：話しかけるだけ（★☆☆☆☆）
   - 演習2：ファイルを渡して議事録作成（★★☆☆☆）
   - 演習3：自分の業務に当てはめる（★★★☆☆）
   - 演習4：複数ファイルの一括処理（★★★★☆）
   - 演習5：CLAUDE.mdの作成（★★★☆☆）

3. **契約・インストールにはトラブル対応込みで18分を確保**
   - 8名を1人でサポートするため、一手順ずつ全員の進捗を確認しながら進める
   - よくあるトラブル（支払い失敗、ダウンロード遅延、ログインエラー）への対処スライドを用意

4. **65分版からの拡張ポイント**
   - 【追加】Cowork vs Claude Code 比較セクション（8分）
   - 【追加】Claude Code 全機能の俯瞰セクション（10分）
   - 【追加】演習4：複数ファイルの一括処理（8分）
   - 【追加】演習5：CLAUDE.mdの作成（5分）
   - 【拡張】契約・インストール作業を独立セクション化し時間を厚く（18分）
   - 【追加】休憩（5分）
   - 【拡張】各演習の時間を1〜2分ずつ延長

5. **前半・後半のバランス**
   - 前半（休憩前）：55分 ── 基礎知識・環境構築・基本操作の体験
   - 後半（休憩後）：60分 ── 応用・自走・カスタマイズ・まとめ
