# Claude Code デスクトップアプリの特徴

- パソコン上のファイルを **直接読み書き** できる
- ファイルを **ドラッグ＆ドロップ** で渡せる
- macOS / Windows 対応
- 複数セッションで並行作業が可能

# /plan ── 「まず計画を見せて」

- 入力欄に **`/plan`** と打ってから指示
- いきなり変更せず **計画だけ** 提示してくれる
- 計画を見て「OK」か「やめる」を選べる

> 大事なファイルを扱うときに使おう

[図：/plan を実行した画面のスクリーンショット]

---
<!-- スライド14: 内容 / 所要時間：1.5分 -->

# /rewind ── 「さっきに戻して」

- 入力欄に **`/rewind`** と打つ
- 過去のチェックポイント一覧が表示
- 会話・ファイル、または両方を巻き戻せる
- **Escキー2回** でも同じ操作ができる

> ゲームの「セーブデータをロード」するイメージ

---
<!-- スライド15: 手順 / 所要時間：3分 -->

2. 計画を承認して実行する
3. `/rewind` で元の状態に戻す

[図：/rewind のチェックポイント選択画面]
